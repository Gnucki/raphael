# Raphaël: Cross-browser vector graphics the easy way - NODE VERSION

Visit the library website for more information: [http://raphaeljs.com](http://raphaeljs.com/)

## Installation

```sh
$ npm install -g gnucki-raphael
```

> The version of gnucki-raphael is mapped on the Raphael's one.
